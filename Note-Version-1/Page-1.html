Release Date: July 02, 2022

Overview

  • We have added the ability to remove card payments on orders
  • New fields "Setup Variable" and "Sample Variable" have been added to the equipment pop up
  • Vendor descripton added on the product level will now show automatically on the PO
  • Information Icons have been added to decoration screen
  • Permissions have been layered in to the new interface
  • Links within the system have been updated to now direct you to the new interface
  • Added an identifier for reorders that have been placed through the Portal.

New Features

Now able to remove card payments on orders

We have added a way to remove payments that were added through a credit card. To accomplish this, you will have to reach out to a support member and identify which order number and payment you would like to remove from the order.

Note: This will not remove the payment from Quickbooks, you will have to remove the payment manually. 

New fields "Setup Variable" and "Sample Variable" added to the equipment pop up
  • Setup Variable will take ___ as input and is used for ____
  • Simple Variable will take ___ as input and is used for ____

When adding equipment in the Production Module, you have two new fields available to you. "Setup Variable" and "Simple Variable". 

Vendor descripton added on the product level will now show automatically on the PO

Bugs and Fixes

  • There was a bug where the Avalara integration produced the incorrect tax value, this has been fixed.
  • The new order form enhancement removed the ability to change the order type from an order to a quote. This functionality has been added back in through the clone process of an order. Now you will be able to clone an order and select quote and credit memo in the popup to convert it back.
  • When users would add items to an order that came from the customer portal, the exiting items qty was effected. This has been corrected.
  • Some orders that came through the customer portal were created with the wrong item quantity. This has been addressed
